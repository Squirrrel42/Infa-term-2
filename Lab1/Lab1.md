### I. Поиск

Функция для поиска полным перебором. При нахождении нужного элемента возвращает индекс элемента в массиве, а если элемент не найден, возвращает -1.
```C++
 int perebor(int arr[], int n, int key)
{
    for(int i=0; i<n; i++){
        if(arr[i] == key){
            return i;
        }
    }
    return -1;
}
```
